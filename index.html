Privacidade e Consentimento

Antes de começar, precisamos do seu consentimento para coletar e armazenar seus dados de acordo com a Lei Geral de Proteção de Dados (LGPD).

Política de Privacidade

1. Quem somos

O Hub Pro é um aplicativo de desenvolvimento profissional para corretores imobiliários, operado pela Hub Litoral — Negócios Imobiliários.

2. Dados coletados

Nome, e-mail, foto de perfil, cidade, especialidade, resultados do Quiz DISC, progresso no Plano 30 Dias, metas e reflexões pessoais.

3. Como usamos

Dados utilizados exclusivamente para personalizar sua experiência no Hub Pro. Não compartilhamos com terceiros para fins comerciais.

4. Armazenamento

Dados armazenados localmente no seu dispositivo via localStorage e opcionalmente sincronizados via Firebase (Google Cloud) com criptografia.

5. Seus direitos (LGPD)

Acesso, correção e exclusão dos dados a qualquer momento. Acesse Perfil → Excluir conta.
